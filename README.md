OVERVIEW
pearl-web is a web-based project that blends minimalist dark aesthetics with soft baby pink highlights. It is designed to be functional, responsive, and visually pleasing.

CERTIFICATION
Verified Project: This repository includes a certification for quality standards and completion.

FEATURES

Aesthetic UI: Dark mode base with baby pink accents.

Responsive Design: Optimized for mobile, tablet, and desktop.

Security: Pre-configured security settings.

Optimized: Fast loading speeds and clean code.

SETUP INSTRUCTIONS

Clone the Repository:
git clone https://github.com/ladypearllamat/pearl-web.git
cd pearl-web

Create Virtual Environment:
python -m venv env
For Windows: .\env\Scripts\activate
For Mac/Linux: source env/bin/activate

Install Dependencies:
pip install -r requirements.txt

Environment Variables:
Create a .env file in the root folder and add your credentials:
Run the Server:
python manage.py migrate
python manage.py runserver
